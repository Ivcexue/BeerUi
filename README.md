## beer-ui
> This is a Svelte Ui Components

## 你需要知道的
- 使用 `pnpm`
- 使用 `Tailwind CSS`
- 使用 `Node 16+`
- 使用 `SCSS`
- 使用 `be-icon` `be-button` 命名方式

## How To Use
svelte & svelte-kit

- `npm i @beerui/beerui -D`
- `npm i sass -D`

`<BeButton type="primary">按钮</BeButton>`
